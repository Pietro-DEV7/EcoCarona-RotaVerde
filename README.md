# EcoCarona & RotaVerde

## Sobre o projeto

O EcoCarona & RotaVerde é uma aplicação web desenvolvida para incentivar a mobilidade sustentável no ambiente escolar. A plataforma permite registrar trajetos realizados a pé, de bicicleta ou por carona, calculando automaticamente os pontos obtidos e a quantidade estimada de CO₂ evitada.

O projeto foi criado com foco na conscientização ambiental e está alinhado à ODS 13 (Ação Contra a Mudança Global do Clima).

---

## Funcionalidades

* Login de usuário.
* Registro de trajetos sustentáveis.
* Cálculo automático de EcoPontos.
* Cálculo de CO₂ evitado.
* Dashboard com métricas atualizadas.
* Ranking escolar.
* Armazenamento local utilizando LocalStorage.

## Como utilizar

### Clonando o repositório

```bash
git clone https://github.com/seu-usuario/ecocarona-rotaverde.git
```

### Acessando o projeto

```bash
cd ecocarona-rotaverde
```

### Executando

Como o projeto não possui dependências externas nem backend, basta abrir o arquivo `index.html` no navegador.

Caso utilize o Visual Studio Code, recomenda-se executar com a extensão **Live Server**.

## Como funciona

1. O usuário realiza o login.
2. Escolhe uma modalidade de transporte.
3. Informa a distância percorrida.
4. O sistema calcula os pontos e o CO₂ evitado.
5. As informações são atualizadas no dashboard e no ranking.
6. Os dados permanecem salvos localmente no navegador.
